# Isekai'd Class
Combines the best features from most classes into one, using Charisma for most abilities.

## Mod Requirements
* Wrath Patches
* Weapon Focus All
* Tabletop Tweaks Base
* Tabletop Tweaks Core
* Tabletop Tweaks Mythic Rework

## Suggested Mods
* ToyBox
* Bubble Buffs
* Cleaner
* One Feat Per Level
* More Bulk Sell
* More Relic Info
* Puzzle Skip
